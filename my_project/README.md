# 贪吃蛇游戏项目

这是一个使用HTML、CSS和JavaScript实现的经典贪吃蛇游戏。

## 项目结构

```
.
├── src/
│   ├── index.html          # 游戏主页面
│   ├── style.css           # 游戏样式
│   └── script.js           # 游戏逻辑
├── init.sh                 # 项目初始化脚本
├── feature_list.json       # 功能列表
└── progress.md             # 项目进度记录
```

## 功能特性

- 🎮 经典贪吃蛇游戏玩法
- 🕹️ 使用键盘方向键控制蛇的移动
- 🍎 食物生成与蛇身增长
- 🧱 碰撞检测（墙壁和自身）
- 📊 得分系统
- ⚙️ 游戏控制（开始、暂停、重新开始）
- 🎨 美观的游戏界面
- 🎯 难度设置（简单、中等、困难）
- 💾 本地存储最高分记录

## 运行方式

1. 确保系统已安装Python
2. 在项目根目录下运行初始化脚本：
   ```bash
   ./init.sh
   ```
3. 在浏览器中打开 http://localhost:8000/src/index.html

## 游戏操作

- 使用 **方向键** 控制蛇的移动
- 按 **空格键** 暂停/继续游戏
- 游戏结束后点击 **重新开始** 按钮开始新游戏

## 技术栈

- HTML5 Canvas
- CSS3
- JavaScript (ES6+)
